Web designing

Web designing is done in four parts -

Discovery phase

In this phase, I'll decide a sitemap for your webpage, sketch a rough, a final draft and a visual image of the website after we've mutually agreed on the contents.

Price: $20 X 10hrs (approx) = $200

Creative phase

In this phase, I'll create a visual design for the website. This includes user experience design, website mockups, and webflow design

Price: $20 X 30hrs (approx) = $400

Development phase

In this phase, I'll create the actual website. This includes HTML, CSS design, animation, making the pages responsive, implementing other features, and including wordpress and additional development. Also I'll test the website in various major browsers.

Price: $20 X 50hrs (approx, can differ largely based on the volume of the site) = $200

In this phase, , and change stuff based on your feedback. Finally, I'll make the website ready for launch. If the website is a dynamic one (i.e. has blogs or uses wordpress or similar services), I can produce a client training video
